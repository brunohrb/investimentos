Open Finance Brasil — Pluggy
🏦
Pluggy
XP Investimentos · Inter · Bradesco · C6 Bank · e mais 200 bancos
Configurar
  1. Crie uma conta grátis em dashboard.pluggy.ai
  2. Crie uma aplicação e copie o Client ID e Client Secret
  3. No Supabase → Edge Functions → Secrets, adicione:
    PLUGGY_CLIENT_ID e PLUGGY_CLIENT_SECRET
  4. Clique em Conectar banco abaixo para autenticar cada banco via Open Finance (OAuth)
Tastytrade
📈
Tastytrade
API REST oficial
Configurar
  1. No Supabase → Edge Functions → Secrets, adicione:
    TASTYTRADE_USERNAME e TASTYTRADE_PASSWORD
  2. Selecione o banco correspondente e informe o número da conta
  3. Salve — o sync vai buscar o NLV (Net Liquidating Value) automaticamente
OKX — Crypto
₿
OKX
API com chave Read-Only
Configurar
  1. No OKX → Assets → API Management → crie uma chave Read-Only
  2. No Supabase → Edge Functions → Secrets, adicione:
    OKX_API_KEY · OKX_API_SECRET · OKX_API_PASSPHRASE
  3. Selecione o banco correspondente e salve
PNC Bank — Plaid
🏛️
Plaid
PNC Bank · e qualquer banco americano
Configurar
  1. Crie uma conta em dashboard.plaid.com e acesse Production
  2. No Supabase → Secrets, adicione:
    PLAID_CLIENT_ID · PLAID_SECRET · PLAID_ENV=production
  3. Selecione o banco e clique em Conectar via Plaid — você será redirecionado para o login do PNC
Após conectar, o access_token é salvo automaticamente e o sync passa a funcionar.
